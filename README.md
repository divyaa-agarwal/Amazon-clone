🛒 Amazon Clone

A front-end clone of the Amazon homepage built using HTML5 and CSS3. This project was created to practice front-end development by recreating a modern e-commerce user interface with clean and organized code.

🌐 Live Demo

Visit the live website:
https://divyaa-agarwal.github.io/Amazon-clone/

✨ Features

- Amazon-inspired homepage
- Navigation bar
- Hero section
- Product showcase sections
- Clean and structured layout
- Responsive design (currently being improved)

🛠️ Technologies Used

- HTML5
- CSS3
- Git
- GitHub
- GitHub Pages

🚀 Getting Started

1. Clone the repository.
2. Open "index.html" in your browser.
